# R223T review ledger summary policy

## Default position

Teacher default draft is the primary reading area. Review ledger summary is secondary and explicitly marked review-only.

## Allowed in ledger summary

- unit_phase_role
- practice_intensity
- teacher_support_density
- event count
- primary pattern list
- component trigger status counts
- screen / sheet / evidence mapping list

## Not allowed in teacher default draft

- practice_pattern_type
- demonstration_type
- micro_practice_type
- appreciation_scaffold_type
- component_trigger
- screen_trigger
- learning_sheet_fields
- evidence_outputs
- new_surface_candidate
- unregistered_do_not_execute

## Safety phrase

Every ledger panel must say:

```text
Review-only metadata. Not executable. Not written back.
```
